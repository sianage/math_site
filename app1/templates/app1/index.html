Mathematics for Computer Science

This website is based on the book Concrete Mathematics: A Foundation for Computer Science. Each chapter is accessible from the side menu and has a page with interactive python code which implements these mathematical concepts.


Chapter 1: Recurrent Problems. These types of problems all have solutions which use the idea of recurrence, in which the solution to each problem depends on the solutions to smaller instances of the same problem.

Chapter 2: Sums. This section develops the notation and general techniques for working with sums.

Chapter 3: Integer Functions. Discrete mathematics primarily uses whole numbers. It is often the case that we need to convert from fractions or real numbers to integers. This section will go over these types of conversions and their interesting properties.

Chapter 4: Number Theory. This section is about the properties of integers.

Chapter 5: Binomial Coefficients. This is an important part of combinatorics. Binomial Coefficients are used to denote the number of possible ways to choose a subset of objects of a given numerosity from a larger set.

Chapter 6: Special Numbers. These are sequences of numbers which occur very frequently in mathematics and have been given special names.

Chapter 7: Generating Functions. The most powerful way to deal with sequences of numbers is to manipulate infinite series that "generate" those sequences. This section explores this concept in-depth.

Chapter 8: Discrete Probability. A discrete probability is one which can be computed by summation instead of integration.

Chapter 9: Asymptotics. When we encounter a sum or recurrence which does not have a closed-form solution, asymptotics can still tell us something about the answer.
